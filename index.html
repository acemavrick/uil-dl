Why UIL-DL

UIL-DL removes the repetitive work of hunting down contest packets. Search once, select what you need, and download everything in one go. A lightweight cache keeps what you already grabbed so repeat downloads are instant and bandwidth‑friendly.

What you can do

  • Filter by subject, contest level, and year with fast column sorting.
  • Batch download PDFs and data files with a single click.
  • Use clear file names that play nicely with shared drives and grading workflows.

How it works

  1. Refresh Info loads the latest catalog maintained for UIL materials.
  2. Pick files, then start the batch—UIL-DL handles concurrency for you.
  3. A local manifest (.cache_manifest.json) tracks what you already have; you can safely refresh or reset it without touching your files.

Privacy

Only anonymous, functional events are measured to improve the app. Delete the ga4_csnt token to opt out at any time.
