Overview

UIL-DL (UIL Downloader) is a tool designed to simplify the process of downloading and managing UIL Academics contest materials. It provides a clean, intuitive interface for filtering and accessing contest files, with features like caching to optimize bandwidth usage and download speed.

Features

  • Search and Filter: easily find contests by subject, level, and year
  • Smart Caching: downloaded files are cached locally to avoid redundant downloads
  • Concurrent Downloads: efficiently manage multiple downloads
  • Responsive UI: clean and easy to use interface

Installation

You can find the latest versions and instructions for macOS and Windows in the releases section. If you prefer building from source, check out the dev branch.

Usage

Downloading Materials

  1. Use the checkboxes to mark which files you want downloaded.
  2. Click “Download Selected” to start downloading all the selected files.

Files are saved to your chosen directory and named like {Contest}_{Year}_{Level}_{filetype}, with spaces replaced by hyphens (e.g. Calculator-Applications_2025_District_pdf.pdf). Quitting the app stops in‑progress downloads.

File Types

  • PDF Packets: packets of the contests, often including the answer key.
  • ZIP Data Files: contest data (e.g., programming), may include answer keys.
  • Other Files: links to files not downloadable by this tool (e.g., Box files).

All files are sourced directly from the official UIL study materials page.

Filtering & Sorting

Filter using the left panel and sort by clicking column headers. Levels follow contest order (Study Packet → Invitational A → Invitational B → District → Region → State).

The Cache

  • Location: files are stored in your downloads directory
  • Manifest: .cache_manifest.json tracks size and timestamps
  • Operations: Refresh Cache updates the index; Reset Cache clears the index without deleting files

Analytics

Basic, anonymous analytics help improve the application. Events are functional (downloads, searches) and never include personal data. To disable analytics, delete the ga4_csnt file in the application support directory.
